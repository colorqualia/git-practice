## Git Basics: Ground level

よく使う Git 基本コマンド

### git init

リポジトリを作成する

### git add

変更ファイルをステージングエリアに追加する

### git commit

変更ファイルをステージングエリアからリポジトリに登録する

### git status

現在の Git ステータスを見る

### git log

コミットログを見る
